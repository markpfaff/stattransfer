New Control over Variable Order and Characteristics


Stat/Transfer 13 has a new, editable, schema format for ASCII delimited files.  If you write a file from any import source into this format, you can then re-order variables, change their names, types, and output widths, assign labels or do anything else you need to get your data in just the format you need.  Then, when you transfer from this format into your chosen final output format, your changes will be applied.  See the documentation for ASCII Schemas for information on how to do this.
