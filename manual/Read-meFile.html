Read-me File
 		   

The installation or Web update procedure may copy a file called read.me, which will be a supplement to the on-line help or manual.  There is a shortcut to the read.me file from the Windows Start menu, in the Stat/Transfer group.


You should check to see if the read.me file exists or has been updated.  If so, you can read it in any editor or word-processor.


We make every effort to keep up with changes in the file formats of popular software and the read.me file will contain the latest information on which versions of these programs are supported. The file will also contain the latest information on other improvements to Stat/Transfer.


You can also get current information about Stat/Transfer by visiting our website at  http://www.stattransfer.com.


You can reach our website from the Stat/Transfer About screen.
