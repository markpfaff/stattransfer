Generating a Stat/Transfer Program
   

You can tell stat transfer to automatically generate a program when you run a transfer.  To do this, click on the Options tab and then select Stat/Transfer Program Generation.  By default, automatic program generation is turned off.


The new program  keeps all of the selections you entered in the dialog boxes. Every option you selected in the Options  dialog box is translated into its equivalent command processor setting. 


This enables you to reproduce your transfer operation either from the Run Program dialog box or as a batch job from the operating system prompt.


It also allows you to precisely document how your transfer was performed.  The program can be viewed or edited in the Run Program dialog box.
