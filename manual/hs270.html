Selecting Members of SAS CPORT and Transport Files
 		   

Whenever you select a SAS CPORT or Transport file as input, Stat/Transfer will display a Member selection line below the input File Specification line of the Transfer dialog box.


The name of the first member will appear on the Member line and, unless you select another one, will be the member of the SAS file used as the input data set by Stat/Transfer. If the data you wish to use are in a different member, click on the control arrow and select the appropriate member from the list that appears.
