The Command Processor

You can run Stat/Transfer without using the user interface, by using the Stat/Transfer command processor.  This allows "batch" jobs to be run for repetitive transfers.


The command processor performs transfers using a copy-like syntax. You can select variables and cases, automatically optimize the output variable types, or change output types manually.
